Personal Note System

Messaging Policy · Privacy Policy · Terms of Use

About This Service

Personal Note System is a private, self-hosted knowledge management tool operated by an individual (the "sole proprietor") for personal use only. The system allows the sole proprietor to send text messages and photos to a dedicated phone number, where they are automatically categorized, analyzed, and stored for later retrieval. The system also provides scheduled reminders and task management via SMS.

This is a personal productivity tool. Only the sole proprietor sends and receives messages through this system. No marketing messages, promotional content, or third-party communications are sent.

Messaging Disclosure

What messages are sent

The system sends the following types of SMS messages to the sole proprietor:

  • Confirmation replies when a note is received and categorized
  • Scheduled reminder notifications at times requested by the sole proprietor
  • Task and to-do confirmations
  • Search results when queried
  • System status updates when requested

Message frequency

Message frequency varies based on the sole proprietor's usage. Typical usage is fewer than 20 messages per day. Messages are only sent in response to the sole proprietor's inbound texts or at times explicitly scheduled by the sole proprietor.

Consent

The sole proprietor is the only user of this system. Consent is established by the sole proprietor's direct action of creating, configuring, and operating the system, and by initiating all inbound conversations. No other individuals are enrolled in or receive messages from this system.

Opt-out

The sole proprietor may stop receiving messages at any time by texting STOP to the system's phone number. To resume, text START.

Help

Text HELP for assistance. For additional support, contact the account owner directly.

Message and data rates

Message and data rates may apply. Consult your mobile carrier for details on your messaging plan.

Privacy Policy

Last updated: February 2026

Information collected

This system processes the following information, all of which is provided directly by the sole proprietor:

  • Text message content sent to the system
  • Photos and images sent to the system
  • Phone number of the sender (sole proprietor only)
  • Timestamps of messages

How information is used

All information is used solely for the purpose of categorizing, storing, and retrieving the sole proprietor's personal notes and managing reminders. Note content is processed by the Anthropic Claude API for analysis and categorization, then stored in a private GitHub repository accessible only to the sole proprietor.

Data sharing

No personal information or note content is sold, shared, or disclosed to any third parties, except as necessary for the technical operation of the system:

  • Twilio — processes SMS messages (subject to Twilio's Privacy Statement)
  • Anthropic — processes note content for AI analysis (subject to Anthropic's Privacy Policy)
  • GitHub — stores notes in a private repository (subject to GitHub's Privacy Statement)

Data retention

Notes are retained indefinitely in the sole proprietor's private repository until manually deleted. The sole proprietor has full control over all stored data and can delete any or all notes at any time.

Security

All data is stored in a private GitHub repository with access restricted to the sole proprietor via personal access token. The server application is secured with environment-based credentials and Twilio request validation.
